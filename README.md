# ABI Burguer V6

Site estático profissional para hamburgueria, pronto para testar no computador, subir no GitHub e publicar na Vercel.

## Como testar no computador

1. Extraia o ZIP.
2. Abra o arquivo `index.html` no navegador.

## Como colocar no GitHub

1. Crie um repositório chamado `abi-burguer-v6`.
2. Envie todos os arquivos desta pasta para o repositório.
3. O `index.html` precisa ficar na raiz do projeto.

## Como publicar na Vercel

1. Acesse https://vercel.com.
2. Clique em `Add New > Project`.
3. Importe o repositório do GitHub.
4. Use:
   - Framework Preset: Other
   - Build Command: vazio
   - Output Directory: vazio
5. Clique em Deploy.
